smartCourtLib:sc-view

Description

在ngView 基础上封装的属性。完成以下功能.

  1. 保存页面切换时是当前页的状态数据如滚动条位置，未提交的表单数据等
  2. 无侵入的为controller.js 的 $scope添加公用方法

用法

    //controller.js
    //回退
    $scope.back()
    $scope.showToast();
    $scope.hideToast()
    $scope.showAlert()
    $scope.hideAlert()

Usage

as attribute
<ANY sc-view>
   ...
</ANY>
