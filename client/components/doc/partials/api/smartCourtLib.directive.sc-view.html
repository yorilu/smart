sc-view
directive in module smartCourtLib

Description

在ngView 基础上封装的属性。完成以下功能.

  1. 为ngView的controller提供生命周期方法，onEnter(),onExit()
  2. 保存页面切换时是当前页的状态数据如$scope.model和滚动条位置
  3. 无侵入的为controller.js 的 $scope添加公用方法

用法

    //controller.js
    //入口方法
    onEnter()
    //出口方法
    onExit()
    //回退
    $scope.back()
    $scope.showToast();
    $scope.hideToast()
    $scope.showAlert()
    $scope.hideAlert()

Usage

as attribute
<ANY sc-view>
   ...
</ANY>

    隐藏Alert

    隐藏toast

  • onEnter()

    每次切换到该页面时执行,为该view Controller的入口方法，可以在此执行业务逻辑代码

    Example

$scope.onEnter = function(){
  //业务逻辑
}

  • onExit(data)

    每次离开该页面时执行，为该view Controller的出口方法，可以在此执行清理工作

   //业务逻辑
